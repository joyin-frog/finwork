# 费用报销管理制度

## 一、总则
规范公司日常费用报销,确保票据合规、审批清晰、入账准确。差旅费另见《差旅费报销管理制度》。

## 二、费用类别与标准
| 类别 | 标准/上限 | 票据要求 |
| --- | --- | --- |
| 办公用品 | 部门月度预算内 | 增值税发票 + 清单 |
| 业务招待 | 单次 ≤ 2000 元 | 发票 + 招待事由/对象 |
| 交通(市内非差旅) | 每月 ≤ 500 元 | 正规发票 |
| 通讯补贴 | 经理级 200 元/月,员工 100 元/月 | 随工资发放,免票 |
| 培训费 | 经审批的预算内 | 发票 + 培训证明 |
| 团建 | 人均 ≤ 200 元/次 | 发票 + 参与名单 |

## 三、审批权限
| 金额 | 审批人 |
| --- | --- |
| ≤ 1000 元 | 部门负责人 |
| 1000–5000 元 | 部门负责人 + 财务经理 |
| 5000–20000 元 | 分管副总 |
| > 20000 元 | 总经理 |

## 四、票据合规要求(报销异常判断依据)
1. 发票抬头须为公司全称,税号正确。
2. 以下情形视为不合规,需标出并退回:
   - 缺少发票或仅有收据。
   - 发票金额与申请金额不一致。
   - 同一发票号重复报销。
   - 超过类别上限且无超标审批。
   - 发票日期超过报销时限(开票后 90 天)。
   - 招待费缺事由/对象。

## 五、流程
申请人提交 → 票据校验 → 逐级审批 → 财务复核入账 → 付款。合规批次生成报销汇总表归档。
