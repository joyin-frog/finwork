# 投资评估与行业标准

## 一、投资评估核心指标
| 指标 | 含义 | 公司参考标准 |
| --- | --- | --- |
| ROI(投资回报率) | 净收益 / 投资额 | ≥ 15% 视为达标 |
| IRR(内部收益率) | 使 NPV=0 的折现率 | ≥ 行业基准 12% |
| NPV(净现值) | 未来现金流折现 − 初始投资 | > 0 方可立项 |
| 投资回收期 | 收回初始投资所需年限 | ≤ 3 年优先 |
| 毛利率 | 毛利 / 收入 | 参考行业中位 |

## 二、折现率(WACC)取定
按资本结构加权,股权成本参考行业风险溢价,债务成本按实际融资利率税后计。基准折现率默认 10%。

## 三、行业标杆参考(用于横向对比)
1. 软件/SaaS:毛利率 70%+,获客回收期 < 12 个月。
2. 制造业:毛利率 20%–35%,资产周转更重。
3. 零售:毛利率 15%–30%,看坪效与周转。
(具体以最新行业研报数据为准,模型中标注数据来源与年份。)

## 四、投资回报分析流程
检索行业标准与历史数据 → 建立现金流模型(收入/成本/税后)→ 计算 ROI/IRR/NPV/回收期 → 敏感性分析(关键假设上下浮动)→ 输出结论与建议(可生成报告)。

## 五、风险提示
模型假设须明确列示;高不确定性项做情景分析;结论为决策参考,不构成承诺。
