# Excel 预览增强 Spec

## 范围

本次只覆盖 Excel 预览体验：

1. 宽表不再被压缩到窄容器里，必须支持横向滚动。
2. 需要能左右查看一行的完整数据。
3. sheet 切换区按工作表样式重做，避免当前简陋按钮感。
4. 预览整体参考表格软件的阅读结构，补充列字母、行号和公式栏。

## 执行目标

- 解析 Excel 时保留工作表范围、列宽、行号和合并单元格信息。
- 表格主体使用 `width: max-content`，外层滚动容器负责横向滚动。
- 顶部 sheet tabs 改成工作表页签风格。
- 表格头部增加列字母，左侧增加行号。
- 公式栏至少展示当前默认聚焦单元格位置和值。

## 自动化测试

1. 预览组件：
   - Excel 渲染包含工作表 tabs、公式栏、横向滚动容器和工作表网格。
   - 解析代码包含列标签和列宽计算逻辑。

2. 样式：
   - Excel 网格滚动容器存在。
   - Excel 表格使用 `width: max-content`。
   - 存在 sheet tabs、公式栏、列头、行头样式。

## 验收目标

- `rtk npx tsc --noEmit` 通过。
- `rtk npm test` 通过。
- `rtk npm run build` 通过。
- 若本地浏览器策略允许，手工验证超宽 Excel 可横向滚动，sheet tabs 视觉更接近表格软件。
